# Lucas Medeiros — SOC Analyst Portfolio

This repository documents my practical learning journey in Cybersecurity, with a focus on SOC operations, Blue Team, SIEM, log analysis, detection engineering, and incident response.

The goal of this portfolio is to demonstrate how I investigate security alerts, write detection logic, map activity to MITRE ATT&CK, classify severity, and document findings in a clear and structured way.
