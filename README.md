# Lucas Medeiros — SOC Analyst Portfolio

This repository documents my practical learning journey in Cybersecurity, with a focus on **SOC operations, Blue Team, SIEM, log analysis, detection engineering, and incident response**.

The goal of this portfolio is to demonstrate how I investigate security alerts, create detection logic, map activity to MITRE ATT&CK, classify severity, and document findings in a clear and structured way.

---

## Focus Areas

- SOC alert triage
- SIEM detection logic
- Windows Event Log analysis
- Sysmon analysis
- Incident response
- MITRE ATT&CK mapping
- Threat intelligence
- Phishing investigation
- Network traffic analysis
- Vulnerability prioritization

---

## Projects

| Project | Description |
|---|---|
| [01 — Brute Force Detection](./01-brute-force-detection) | Investigation of multiple failed login attempts followed by a successful login, including detection logic, an investigation report, and MITRE ATT&CK mapping |
| [02 — Suspicious PowerShell Execution](./02-suspicious-powershell) | Analysis of suspicious PowerShell activity using Sysmon logs, command-line behavior, detection logic, and MITRE ATT&CK mapping |
| [03 — Phishing Investigation](./03-phishing-investigation) | Investigation of suspicious emails, links, attachments, sender reputation, authentication results, and indicators of compromise |
| [04 — C2 Beaconing Analysis](./04-c2-beaconing-analysis) | Analysis of repeated outbound network communication that may indicate command and control activity |
| [05 — Vulnerability Prioritization](./05-vulnerability-prioritization) | Prioritization of vulnerabilities using CVSS, exploitability, asset criticality, exposure, and business context |
